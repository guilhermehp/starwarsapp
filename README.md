# Star Wars App

## Introduction
The **Star Wars App** is a dynamic Android application that interfaces with the Star Wars API to display items from six categories: characters, films, planets, species, starships, and vehicles. The main screen allows users to switch between these categories using a tab bar and view details about each item by navigating to a detailed screen. Users can customize the app theme, manage favorites via a heart icon, and navigate easily with a home floating button. Favorites can be reviewed and filtered in a dedicated screen.

## Features
- **Customizable UI:** Change the app theme with a simple click.
- **Favorites Management:** Add or remove items from the favorites list.
- **Rich Navigation:** Browse detailed information through interconnected data fields.
- **Persistent Data:** Favorites and settings are stored locally.

## Development Environment

- Android Gradle Plugin Version 8.3.2

## Key Technologies
- **Retrofit:** For API consumption.
- **Room:** For local database storage.
- **DataStore:** For UI style settings.
- **Jetpack Compose:** For UI design and animations.
- **Jetpack Navigation:** For handling app navigation.
- **Kotlin Flow and Coroutines:** For asynchronous data handling.
- **StateFlows:** For state management in UI.
- **Hilt:** For dependency injection.
- **JUnit and Compose Test:** For unit and UI testing.
